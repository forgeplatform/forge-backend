# Legacy UI

This directory contains the legacy AWX PatternFly UI that ships with the upstream AWX codebase.

**The active Forge frontend is in a separate repository:** [forge-frontend](https://github.com/forgeplatform/forge-frontend)

The legacy UI code is retained for backward compatibility with AWX internals (management commands, static file collection) but is not actively developed or used in production Forge deployments.

For frontend development, see the `forge-frontend` repository.
